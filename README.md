## `❯ whoami`

Backend engineer building open-source Java frameworks and developer tooling. I like clean abstractions, dependency-light libraries, and infrastructure that stays readable under pressure.

Lately I turn repetitive backend work into reusable starters and generators, and wire LLMs into real Java services.
